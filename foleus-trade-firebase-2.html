🔥 FIREBASE SETUP
Follow these steps to enable cloud sync
1
Create Firebase Project Go to console.firebase.google.com → Click "Add project" → Enter a name → Continue
2
Enable Authentication In your project → Authentication → Sign-in method → Enable Email/Password and Google
3
Create Firestore Database Firestore Database → Create database → Choose Start in test mode → Select region → Done
4
Get Your Config Project Settings (⚙) → General → Your apps → Web app (</>) → Register app → Copy the firebaseConfig object
5
Paste Config into this file Open this HTML file in a text editor → Find const firebaseConfig = { → Replace with your copied config → Save & reopen
